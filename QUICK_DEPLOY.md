# 🚀 ONE-CLICK DEPLOY TO RENDER

## Your site is ready! Just follow these 3 simple steps:

### Step 1: Go to Render Deploy Page
**Click here:** https://render.com/deploy?repo=https://github.com/aljannatulfirdous2007-rgb/ecommerce_ai_client2

OR manually:
1. Visit https://dashboard.render.com
2. Click **"New +"** button
3. Select **"Static Site"**

---

### Step 2: Connect Your Repository
- Choose: **"Deploy from Git repository"**
- Select repository: `aljannatulfirdous2007-rgb/ecommerce_ai_client2`
- Branch: `main`

---

### Step 3: Configure Build Settings ⚙️

**IMPORTANT - Use these EXACT settings:**

```
Name: al-firdous-luxe
Root Directory: client
Build Command: npm install && npm run build
Publish Directory: dist
```

**Advanced Settings (optional):**
- Node Version: 20

---

### Step 4: Deploy! 🎉
- Click **"Create Static Site"**
- Wait 2-3 minutes for build
- Your site will be LIVE at: `https://al-firdous-luxe.onrender.com`
